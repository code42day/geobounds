# geobounds

Checks if point is in bounds.

## Install

```sh
$ npm install --save geobounds
```

## Usage

```js
var geobounds = require('geobounds');
var bounds = geobounds([
  [[-1, 0], [3, 5]],
  [[100, 20], [115, 80]]
]);

bounds([1, 1]); // true
bounds([4, 1]); // false

```
